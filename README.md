# 🚌 CITY BUS — مسارات الحافلات في مكناس

> الدليل الرقمي التفاعلي لخطوط النقل الحضري بمدينة مكناس، المغرب.

---

## معاينة المشروع / Preview

| الصفحة | الوصف |
|--------|-------|
| `index.html` | الخريطة التفاعلية — ابحث عن حافلة واعرض مسارها |
| `all_buses.html` | قائمة جميع الخطوط المتاحة |
| `about.html` | معلومات عن المشروع والفريق |
| `report.html` | صفحة اقتراح تعديل (قريباً) |

---

## هيكل المشروع / Project Structure

```
city-bus/
```

## التقنيات المستخدمة / Tech Stack

| التقنية | الاستخدام |
|---------|-----------|
| [Leaflet.js](https://leafletjs.com/) | عرض الخرائط التفاعلية |
| [OSRM](https://project-osrm.org/) | حساب مسار الطريق الحقيقي |
| [Esri World Imagery](https://www.arcgis.com/) | طبقة الخريطة (صور الأقمار الصناعية) |
| [Font Awesome 6](https://fontawesome.com/) | الأيقونات |
| HTML / CSS / Vanilla JS | البنية والتنسيق والمنطق |

لا يوجد أي framework أو build tool — المشروع يعمل مباشرة في المتصفح.
